# .NET Core OTP (One-Time PIN) API

A clean, open-source, and production-ready .NET 8 Web API for generating, storing, and validating One-Time PINs (OTPs). 

## Getting Started

1. Clone the repository.
2. Open `OtpService.sln` in Visual Studio 2022.
3. Set `OtpService.Api` as the startup project.
4. Press `F5` to run. Swagger UI will open automatically at `https://localhost:<port>/swagger`.

## Running Tests
Navigate to the `tests/OtpService.Tests` directory and run:
`dotnet test`
